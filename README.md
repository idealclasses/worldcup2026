# worldcup2026

A simple static website that displays the World Cup 2026 match schedule, completed fixtures, and winners.

## Files

- `index.html` — main website page
- `styles.css` — site styling
- `script.js` — schedule data and interactive filtering

## Run locally

1. Open `index.html` in a browser directly, or
2. Start a local server from the repository root:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000` in your browser.
